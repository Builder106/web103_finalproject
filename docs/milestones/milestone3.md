## Reflection

### 1. What went well during this unit?

Our team made strong progress on the core Study Sprint Tracker experience. We successfully connected the React frontend to the Express backend, built the goal progress dashboard, and implemented the timer/session logging flow so users can track sessions without losing context.

### 2. What were some challenges your group faced in this unit?

The biggest challenge was syncing the timer/session UI with the database and keeping the goal progress state consistent across the detail view and dashboard. We also had to coordinate GitHub issue creation and project board updates while simultaneously writing backend routes and frontend components.

### Did you finish all of your tasks in your sprint plan for this week? If you did not finish all of the planned tasks, how would you prioritize the remaining tasks on your list?

We completed the essential sprint plan items for Milestone 3: project management setup, feature issue tracking, and core feature development. For any remaining tasks, we would prioritize bug fixes and data validation first, then polish the UI and add any remaining stretch features.
