# StudyQuill

StudyQuill is now an exact copy of the `web103_finalproject/prototype` frontend prototype.

## What’s included

- React + Vite frontend copied directly from the prototype
- Prototype app structure under `src/app`
- Prototype routing, dashboard, and goal detail UI

## Local development

1. `cd StudyQuill`
2. `npm install`
3. `npm run dev`

## Notes

- This folder now matches the prototype design instead of the earlier custom layout.
- The prototype is front-end focused; backend integration can be implemented on top of this app.
